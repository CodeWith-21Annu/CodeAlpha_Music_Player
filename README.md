CodeAlpha Music Player

A responsive and interactive Music Player built using HTML, CSS, and JavaScript as part of the CodeAlpha Web Development Internship.

Features:
- Play and Pause songs
- Next and Previous controls
- Progress Bar
- Volume Control
- Playlist Support
- Responsive and Attractive UI

Technologies Used:
- HTML
- CSS
- JavaScript
